# TwitterAPI-React
This is the React application that uses TwitterAPI for fetching tweets of a user.

## Technologies Used
- React
- Next.js
- Redux
- Bootstrap

## Installing the Packages

```bash
npm install
```

## Run the Application

```bash
npm run dev
```
